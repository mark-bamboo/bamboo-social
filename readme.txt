== Description ==

This plugin provides a widget and a shortcode for generating social media icons that link to the relevent social media accounts.

Usage

Click on Bamboo Social in the dashboard menu and fill in the links to the social media accounts you would like icons for.

Insert the Bamboo Social widget in the required widget area.

Alternatively you can insert the shortcode [bamboo-social] where ever you would like the icon to appear.

== Installation ==

1. Upload `bamboo-social` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
